# Django Practice Project

## Overview
This project documents my journey learning Django by following the comprehensive beginner's guide series from SimpleisBetterThanComplex.

## Learning Resource
I am working through the Django Beginners Guide available at:
**https://simpleisbetterthancomplex.com/series/beginners-guide/1.11/**

This series provides excellent foundational knowledge for Django development, covering everything from basic setup to more advanced concepts.

## Project Goals
- Understand Django fundamentals and project structure
- Learn how to create models, views, and templates
- Practice building URL routing and forms
- Develop skills in database design and ORM usage
- Build full-featured web applications with Django

## Progress
This repository contains the practical exercises and projects completed while following the tutorial series.

## Getting Started
To run this project:
1. Ensure Django is installed
2. Set up a virtual environment
3. Install dependencies from `requirements.txt`
4. Run migrations: `python manage.py migrate`
5. Start the development server: `python manage.py runserver`
